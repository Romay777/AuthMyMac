# Security Policy

AuthMyMac is a local-only authenticator. Please do not include TOTP secrets,
provisioning URIs, migration payloads, or Keychain data in an issue or pull
request.

For a suspected security vulnerability, use the repository's
[private vulnerability reporting](https://github.com/Romay777/AuthMyMac/security/advisories/new)
instead of opening a public issue. Include enough information to reproduce the
problem without sharing real credentials.

See [the security boundaries](Docs/SECURITY.md) for the application's storage,
logging, permission, and export guarantees.
